# 🧩 Sudoku Game (React + TypeScript)

Interactive Sudoku game built with React, TypeScript and Vite.

---

## 🚀 Features

- Sudoku gameplay
- Timer
- Difficulty levels
- Game results history
- Result modal (win/lose/continue)
- Cookie consent (GDPR)
- LocalStorage saving
- Storybook UI documentation
- TypeDoc generated documentation

---

## 🛠 Tech Stack

- React
- TypeScript
- Vite
- CSS Modules
- Storybook
- TypeDoc
- react-cookie-consent

---

## 📦 Installation

```bash
npm install
```

## ▶️ Run project

```bash
npm run dev
```

## 🏗 Build

```bash
npm run build
```

## 📚 Storybook

```bash
npm run storybook
```

## 📖 Documentation

```bash
npm run docs
```

---

## 🍪 GDPR / Cookies

- Stores game progress locally
- Saves user preferences
- No server data collection
