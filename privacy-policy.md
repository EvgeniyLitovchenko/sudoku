# Privacy Policy

## Overview

This document explains how the Sudoku application handles user data and privacy.

The application is developed as a learning project and is intended for demonstration and educational use only.

---

## Data Usage

The Sudoku app does not collect or process personal information such as:

- Full name
- Email address
- Contact details
- Payment information

All interactions occur locally within the user's browser.

---

## Local Storage

To improve user experience, the application may store limited data in the browser, including:

- Current game progress
- Selected difficulty level
- User preferences

This data is stored using browser technologies such as Local Storage and is never sent to external servers.

---

## Cookies

The application may display a cookie consent notification.

Cookies (if enabled) are used only for:
- remembering user consent
- storing basic preferences

No tracking or advertising cookies are used.

---

## Third-Party Libraries

The project may include third-party libraries (for UI or functionality).  
These libraries operate within the application and do not independently collect personal user data.

---

## User Control

Users have full control over their data. You can:

- Clear Local Storage via browser settings
- Remove cookies at any time
- Stop using the application without any data retention

---

## GDPR Considerations

The application follows basic GDPR principles:

- No personal data collection
- Transparent use of browser storage
- User awareness via cookie notification

---

## Policy Updates

This Privacy Policy may be updated during development.  
Any changes will be reflected in this file.
